# Generell melding
Versjon 1.0 av [generellMelding](1.0/generellMelding.xsd) brukes i dag mellom påtalemyndigheten og domstolene. Denne erstatter den forrige [meldingStraffesak](../melding/README.md)
# Bruk av generellMelding
| Avsender    | Mottaker                            |kommentar|
|:------------|:------------------------------------|:---|
| Påtale      | Tingrett, lagmannsrett, Høyesterett |Beramming. ønsker ny bevisoppgave/sluttinnlegg/saksdokumenter
| Tingrett, lagmannsrett, Høyesterett | Påtale                              |Bevisoppgave, sluttinnlegg, ...
| Kriminalomsorgen | Domstolene|Brukes i bruddsaker|
